### Double DQN Navigation project using Unity environment (<a href="https://github.com/Zaharah/DeepRL-DQN-unity-navigation"> Code </a>)
An agent is trained to navigate in a square world to collect yellow bananas in a discrete action space.  The gif shows an `agent in action`  trained with Double DQN algorithm for 1000 episodes. The goal is to collect as many yellow bananas as possible while avoiding blue bananas.

### Deep Deterministic Policy Gradients for continuous control (<a href="https://github.com/Zaharah/DeepRL-DDPG-Reacher-Continuous"> Code </a>)
An RL agent is trained to maintain the moving ball at the target location. The environment has contineous action space corresponding to torque applicable to two joints.  A DDPG algorithm is implemented to solve the environment.

### Multi-agent continuous control for competition and collaboration (<a href="https://github.com/Zaharah/DeepRL-MADDPG-Tennis-Unity"> Code </a>)
The environment has multiple agents seeking to compete and collaborate for tennis play. The multi-agnet version of DDPG is implemented to train the agents in contineous action space.
