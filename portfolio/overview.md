### Double DQN Navigation project using Unity environment
An agent is trained to navigate in a square world to collect yellow bananas.  The gif shows an `agent in action`  trained with Double DQN algorithm for 1000 episodes. The goal is to collect as many yellow bananas as possible while avoiding blue bananas.
